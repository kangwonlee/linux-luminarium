지금까지 우리는 여러 가지 방식으로 명령을 실행해 보았습니다:

 ● 절대 경로를 통해 실행하는 방법(예: /challenge/run)

 ● 상대 경로를 통해 실행하는 방법(예: ./run)

 ● 명령어 이름만 입력해 실행하는 방법(예: ls)

앞의 두 가지, 즉 절대 경로와 상대 경로 방식은 비교적 명확합니다. run 파일은 /challenge 디렉터리에 있고, 두 경우 모두 그 파일을 직접 가리킵니다
(물론 상대 경로를 사용할 때는 현재 작업 디렉터리가 /challenge여야 합니다).
